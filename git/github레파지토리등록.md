# GIT HUB 레파지토리 등록

## git 설정

- echo "# Project Name ~~" >> README.md
- git init
- git add README.md
- git commit -m "first commit"
- git branch -M main --> 브런지를 master에서 main으로 변경
- git hub 사이트에 저장소 생성
- git remote add origin http://github.com/XXXXX~~XXX.git --> git hub 저장소 주소
- git push -u origin main
